Prepare a CSV File

Some columns are required, some are optional.

warning

Some of these headers changed with version 2023.2! Please check your CSV headers.

Required columnsâ€‹

  • Number
  • Type
  • Name
NumberTypeName
12startCue 12 GO

Optional Columnsâ€‹

  • Notes
  • Follow
    • 0 - No Follow
    • 1 - Auto-Continue
    • 2 - Auto-Follow
  • Color (Options)
  • Target
  • File Target
  • Columns for Group cues.
    • Group Mode
      • Options:
        • 0 - List
        • 1 - Start first and enter
        • 2 - Start first
        • 3 - Timeline
        • 4 - Start random
        • 6 - Playlist (This is not a typo, 6 is the command for Playlist type per QLab Docs)
  • Columns for Text cues. As of v.2023.3
    • Text
  • Columns available for "fade" cue type:
    • Stop Target When Done
      • "True" or "False"
    • Fade Opacity
      • (ONLY 0 or 1 per QLab docs)
      • Note: Also activates the checkbox for the fade.
  • Columns available for "video" cue type:
    • Stage Number (In order of list in video outputs setting)
note

Stages are in QLab 5 only.

  • Columns available for "midi" cue type:
    • MIDI Q Number
    • MIDI Device ID
    • MIDI Message Type
      • 1 - MIDI Voice Message ("Musical MIDI")
      • 2 - MIDI Show Control Message (MSC)
      • 3 - MIDI SysEx Message
    • MIDI Control Number
    • MIDI Control Value
    • MIDI Patch Channel
    • MIDI Patch Number
    • MIDI Q List
    • Midi Raw String
      • For SysEx Messages
    • MIDI Command Format (Options)
    • MIDI Command (Options)
  • Columns available for "network" cue type:
    • QLab 5
      • Network Patch Number
      • Network Patch Channel
      • Custom String
    • QLab 4
      • Message Type (Options)
      • OSC Cue Number (Only if using QLab Message Type)

      • Command

        • For QLab Messages (Options)

        • For an OSC message, you may now include a raw string in this column
