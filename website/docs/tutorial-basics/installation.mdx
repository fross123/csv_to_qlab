As requested after you agree to the terms of use, move the application to the Applications folder on your computer.

![Move the App](/img/tutorial/move-app-screenshot.png)

## Eject disk image

Open finder, and eject the disk image. You will not need this anymore.

You will see the item below on the left sidebar of the finder under a heading "locations". Click the eject button to eject the installer.

![Eject Disk Image](/img/tutorial/eject-screenshot.png)


## Open Application

Go to your Application folder. Often found in the Finder sidebar. Open CSV to QLab.


## Confirm Security Settings

Apple only verifies applications under their "Developer" program, which has a yearly fee of $99. This is a side project, and therefore does not have a developer key attached. The code is public, you may confirm that the software is not malicious by going to the [GitHub](https://www.github.com/fross123/csv_to_qlab/) repository.

More info on [Apple Certificates](https://developer.apple.com/support/certificates/).

:::tip
You will only need to do these steps one time per-download.
:::

## Command-Line Interface (CLI)

For automation, scripting, or cross-platform use, install the CLI with pip:

```bash
# Clone the repository
git clone https://github.com/fross123/csv_to_qlab.git
cd csv_to_qlab

# Install
pip install .
```

**Next Steps:**
- Continue to [Prepare a CSV File](/docs/tutorial-basics/prepare-csv-file) to format your data
